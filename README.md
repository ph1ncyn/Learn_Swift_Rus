### Чтобы начать кодить на Swift нужно установить приложение XCode на macOS.
Дальше создаём файл:
1. Откройте Xcode и выберите в меню: **File > New > Playground**.
2. Выберите шаблон **Blank** (пустой) для iOS или macOS.
3. **Центральная часть окна** — это редактор кода, куда вы вписываете команды.
4. **Панель справа** — это панель результатов, где мгновенно отображается итог выполнения каждой строки.
5. Чтобы запустить код, нажмите на **кнопку Play** (треугольник) внизу или слева от строки

### Поговорим про переменные как вписываются и как работают

```Swift
// Константа (значение не меняется)
let name = "Alex" 

// Переменная (значение можно менять)
var score = 0 
score = 10 // Теперь в переменной значение 10 [6-9]

// Типы данных (Swift может определять их сам, но можно указать явно)
let age: Int = 25          // Целое число [10]
let price: Double = 99.99  // Число с плавающей точкой [11]
let isReady: Bool = true   // Логический тип (да/нет) [12]
```

### Дальше поговорим про логику и циклы

```Swift
let userAge = 20

if userAge >= 18 {
    print("Доступ разрешен") // Выведет текст в консоль [13, 14]
} else {
    print("Доступ запрещен") [14]
}
```

```Swift
let fruits = ["Яблоко", "Банан", "Груша"]

for fruit in fruits {
    print("Я люблю \(fruit)") // Выведет фразу для каждого фрукта [17, 18]
}
```

### Дальше ОПЦИОНАЛЫ (Безопасность)

В Swift переменная может не иметь значения (**nil**). Чтобы программа не «упала» при обращении к пустоте, используйте **Optional Binding** (`if let`)

Пример:
```Swift
var optionalName: String? = "Ivan" // Знак вопроса значит, что тут может быть nil

if let definiteName = optionalName {
    print("Привет, \(definiteName)") // Код сработает, только если значение есть
} else {
    print("Имя не указано")
}
```

### func

Используйте функции, чтобы сгруппировать код, который вы планируете вызывать часто
**Как вписывать:** Объявите функцию с помощью `func`, а затем вызовите её по имени ниже

```Swift
// Объявление
func sayHello(userName: String) {
    print("Привет, \(userName)!") [30, 31]
}

// Вызов
sayHello(userName: "Дмитрий") [30, 32]
```

### SwiftUI

Для создания кнопок и текста в приложении используется декларативный стиль. Этот код пишется внутри структуры `ContentView` в файле проекта (не в Playground)

**Фрагмент кода для создания счетчика:**
```Swift
import SwiftUI

struct ContentView: View {
    @State private var counter = 0 // Переменная состояния для отслеживания значения

    var body: some View {
        VStack { // Вертикальный стек (элементы друг под другом)
            Text("Счётчик: \(counter)")
                .font(.largeTitle) // Модификатор шрифта
            
            Button("Увеличить") {
                counter += 1 // Действие при нажатии
            }
            .padding() // Отступ
            .background(Color.blue) // Синий фон кнопки
            .foregroundColor(.white) // Белый текст
            .cornerRadius(10) // Скругление углов
        }
    }
}
```

**Как это работает:** Вы меняете код в левой части Xcode, а в правой части (**Canvas**) мгновенно видите результат. Если вы меняете значение `counter`, SwiftUI сам перерисовывает экран
